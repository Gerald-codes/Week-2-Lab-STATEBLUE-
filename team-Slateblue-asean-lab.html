Using na.rm = TRUE assumes that it is acceptable to exclude the missing production values from the total. 3.5 Add a Country-Code Column

  "Myanmar" ~ "MMR",
  "Philippines" ~ "PHL",
  "Singapore" ~ "SGP",
  "Thailand" ~ "THA",

Explanation
We added an ISO 3-letter country-code column to make the country names easier to reference. We checked that the new iso3 column contained no missing values and confirmed that there were ten unique country codes, matching the ten ASEAN countries in the dataset.
